# MMAssistant

Десктопное приложение на Python для автоматической обработки аудиоконтента с использованием Whisper ASR и LLM.

## Назначение проекта

MMAssistant предназначен для русскоязычных пользователей, работающих с аудиоконтентом (исследователи, журналисты, студенты). Приложение предоставляет инструменты для записи аудио, распознавания речи, обработки текста через LLM и управления JSON стенограммами.

## Ключевые возможности

- **Запись аудио**: Системный звук + микрофон с микшированием
- **Распознавание речи**: Интеграция с Whisper ASR API
- **Обработка текста**: Работа с LLM API (OpenAI, Yandex GPT и другие)
- **Управление стенограммами**: Работа с JSON файлами, сегментация, фильтрация
- **Безопасное хранение**: API ключи хранятся в Windows Credential Manager
- **Гибкие настройки**: Конфигурация через графический интерфейс

## Основные компоненты

- **`gui.py`** - Главный графический интерфейс (Tkinter)
- **`audio_recorder.py`** - Модуль записи и обработки аудио
- **`request_processor.py`** - Обработчик запросов к ASR и LLM API
- **`settings_module.py`** - Управление настройками и конфигурацией
- **`config_manager.py`** - Работа с конфигурационными файлами

## Быстрый старт

Подробная инструкция по запуску доступна в [QUICK_START.md](QUICK_START.md).

**Рекомендуемый способ**:
1. Запустите `setup.bat` для установки зависимостей
2. Запустите `start.bat` для запуска приложения

## Установка и запуск

### Зависимости
Основные зависимости перечислены в [requirements.txt](requirements.txt):
- `requests>=2.31.0`
- `numpy>=1.24.0`
- `soundfile>=0.12.0`
- `pyaudiowpatch>=0.2.0`
- `keyring>=24.3.0`

### Варианты запуска
1. **Полная установка**: `setup.bat` → `start.bat`
2. **Упрощенный запуск**: `run_simple.bat` (минимальные зависимости)
3. **Ручной запуск**:
   ```bash
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   python gui.py
   ```
