# Инструкция по запуску MMAssistant

## Вариант 1: Полная установка (рекомендуется)

1. **Запустите `setup.bat`** - двойной клик по файлу
   - Создаст виртуальное окружение
   - Установит все зависимости
   - Может занять несколько минут

2. **Запустите `start.bat`** - двойной клик по файлу
   - Активирует виртуальное окружение
   - Запустит приложение

## Вариант 2: Упрощенный запуск (если setup.bat не работает)

1. **Убедитесь, что установлен Python 3.8+**
   - Проверьте: `python --version` в командной строке

2. **Запустите `run_simple.bat`**
   - Установит минимальные зависимости (requests, keyring)
   - Запустит приложение
   - **Внимание**: функции аудио могут не работать без numpy

## Вариант 3: Ручной запуск

1. Откройте командную строку в папке с проектом
2. Создайте виртуальное окружение:
   ```
   python -m venv venv
   ```
3. Активируйте его:
   ```
   venv\Scripts\activate
   ```
4. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
5. Запустите приложение:
   ```
   python gui.py
   ```

## Решение проблем

### Проблема: "Виртуальное окружение не найдено"
**Решение**: Запустите `setup.bat` перед `start.bat`

### Проблема: Ошибки при установке зависимостей
**Причина**: Проблемы с подключением к интернету
**Решение**:
1. Проверьте подключение к интернету
2. Попробуйте запустить `run_simple.bat`
3. Или установите зависимости вручную:
   ```
   pip install requests keyring
   ```

### Проблема: "Python не найден"
**Решение**: Установите Python 3.8+ и добавьте в PATH

## Проверка работы

После запуска приложения:
1. Нажмите кнопку **"Настройки"** в главном окне
2. Проверьте работу всех трех вкладок:
   - **LLM** - введите API ключ (будет сохранен в Windows Credential Manager)
   - **Whisper** - настройте endpoint
   - **Папки** - настройте пути

## Важные изменения

### Безопасное хранение ключей
- API ключи больше не хранятся в `config.json`
- Используется Windows Credential Manager
- При первом запуске нужно ввести ключ заново

### Новое меню настроек
- Кнопка "Сохранить настройки" удалена из главного окна
- Все настройки перенесены в отдельное меню
- Настройки сохраняются автоматически при закрытии окна
